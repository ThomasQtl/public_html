Game of Life

Basic game

The Game of Life according to Conway's Life Rule (B3/S23).
It is possible to speed up or slow down the spread with the right and left arrows.
The mode between delimited and toric can be modified directly with the M key.
If we arrive in a stable state, the console displays "Stable state".

User Interaction

In addition, it is possible to pause the game with the space bar, to draw its diagram at any time (pause or not) using the mouse.
But also save a configuration in a text file with the S key and load a configuration with the C key.

Other setup

Finally, it is possible to change the rules of the game in the data.h file which simply contains two tables of size 9 containing the rules of birth and survival.
