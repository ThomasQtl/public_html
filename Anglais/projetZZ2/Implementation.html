Main programme

LabVIEW 17 allows you to do graphical programming.
Here, it is the principal program which contains several sub-programs which one calls "sub-vi".

Sound acquisition

This sub-vi allows the acquisition of sound by putting the different acquisitions in a queue for processing.

Sound processing

This sub-vi is used to process the sound.
Display of the temporal sound.
Calculation & display of the DSP.
Averaging of values.

Averaging

Averaging makes it possible to reduce the number of final values to drive an electrolysis system for copper etching that we did not treat.
