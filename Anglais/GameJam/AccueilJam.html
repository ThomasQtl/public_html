Game Jam

Presentation

The ISIMA Pixel club, where I am secretary, organizes Game Jams.
This is an event that spans a full weekend where you have to create a video game.
This is done by team or individually in the language or with the game engine that you want.
A theme is given at each start of Game Jam. During the weekend, we have to think about our game, learn/discover a language or a game engine if we don't already know it, make or search sprites...
